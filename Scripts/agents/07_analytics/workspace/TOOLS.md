# TOOLS — Analytics Agent

## Pfade
- Input Published: ~/Nous/Published/
- Output Reports: ~/Nous/Scripts/output/analytics/
- Output Strategy Input: ~/Nous/Scripts/output/analytics/strategy_input.json

## API Keys (aus ~/Nous/.env)
META_ACCESS_TOKEN → Instagram Insights
X_API_KEY → Twitter Analytics
TIKTOK_ACCESS_TOKEN → TikTok Analytics

## Report-Dateien
analytics_YYYY-MM-DD.json  → Vollständiger Report
analytics_YYYY-MM-DD.md    → Lesbare Zusammenfassung für WhatsApp
