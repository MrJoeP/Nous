## What This Agent Does

After approval, automatically posts to TikTok:
- Videos (clips with caption)
- Direct post or save as draft

---

## ⚠️ Important Notes

- Requires a **TikTok Developer Account**
- App must be reviewed and approved (can take 1–2 weeks)
- Only video posts are possible — no image-only posting
- At least one video must already be on the account

---

## Prerequisites

### 1. TikTok for Developers
- Go to: developers.tiktok.com
- Create account with @join.nous TikTok account
- "Create App" → App name: Nous Publisher

### 2. Activate Content Posting API
- Developer Portal: Products → Content Posting API
- Request scopes: `video.publish` and `video.upload`
- Submit app for review

### 3. OAuth Flow

```
Authorization URL:
https://www.tiktok.com/v2/auth/authorize
  ?client_key=YOUR_CLIENT_KEY
  &scope=video.publish,video.upload
  &response_type=code
  &redirect_uri=YOUR_REDIRECT_URI
  &state=random_string
```

### 4. Get Access Token

```bash
curl -X POST "https://open.tiktokapis.com/v2/oauth/token/" \
  -H "Content-Type: application/x-www-form-urlencoded" \
  -d "client_key=YOUR_KEY" \
  -d "client_secret=YOUR_SECRET" \
  -d "code=AUTH_CODE" \
  -d "grant_type=authorization_code" \
  -d "redirect_uri=REDIRECT_URI"
```

## Post a Video (2-Step Process)

### Step 1: Initialize upload

```bash
curl -X POST "https://open.tiktokapis.com/v2/post/publish/video/init/" \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN" \
  -H "Content-Type: application/json; charset=UTF-8" \
  -d '{
    "post_info": {
      "title": "YOUR CAPTION",
      "privacy_level": "PUBLIC_TO_EVERYONE",
      "disable_duet": false,
      "disable_comment": false,
      "disable_stitch": false
    },
    "source_info": {
      "source": "PULL_FROM_URL",
      "video_url": "https://YOUR-VIDEO-URL",
      "video_cover_timestamp_ms": 1000
    }
  }'
```

### Step 2: Check status

```bash
curl -X POST "https://open.tiktokapis.com/v2/post/publish/status/fetch/" \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN" \
  -d '{"publish_id": "ID_FROM_STEP_1"}'
```

## Token Refresh (critical!)

TikTok Access Token expires after **24 hours**.  
Refresh Token expires after **365 days**.

```bash
# Refresh daily via cron:
curl -X POST "https://open.tiktokapis.com/v2/oauth/token/" \
  -d "client_key=KEY" \
  -d "client_secret=SECRET" \
  -d "grant_type=refresh_token" \
  -d "refresh_token=REFRESH_TOKEN"
```
